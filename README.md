   I. About:
	This is an RDBMS implementation to track television shows and all their
respective information. Including series, seasons, episodes, characters, 
related people, and providers.
	
  II. Requirements:
	Database software: Oracle 11gR2
	Schema: tvdb/tvdb
	Java SDK 1.7.0_u11
